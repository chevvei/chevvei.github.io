更改底部的power强力驱动
找到zh-CN.yml文件中的footer :，更改powered对应的文字
找到footer.swig文件，查找一下代码块

set next_site

‘footer.powered’ 索引到zh-CN文件中的footer ,powered对应的文字
next_url里面是文字对应的网址

这段设置的是&的前一部分
‘footer.powered’, next_url(‘https://chevvei.github.io/img/me.jpg', ‘陈vv’, {class: ‘theme-link’})

这段设置的是&的后一部分
next_url(next_site, ‘我数火车的阳台’, {class: ‘theme-link’})

1 安装
npm install –save hexo-tag-aplayer

2 添加 MetingJS
修改 hexo 根目录下的 _config.yml ,在 _config.yml 里的任意位置添加：

aplayer:
meting: true
